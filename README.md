Configuration
- Create virtual environment for the project
- Install the requirement by pip install -r requirement.txt
- run the download_image.py to download the pokemon image
- run app.py then open 'http://127.0.0.1:5000/'
- to get the pokemon data open 'http://127.0.0.1:5000/get_pokemon'
- get back to 'http://127.0.0.1:5000/' to see pokemon data
